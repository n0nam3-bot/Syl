📊 Google Apps Script (Required for data)
First-time setup:
1. Go to script.google.com → New project
2. Paste the Code.gs file contents → change SS_ID to your Google Sheet ID
3. Deploy → New deployment → Web App → Execute as: Me → Access: Anyone
4. Copy the /exec URL here → Save → click Initialize Data Sheet
5. Wait ~60 seconds for GOOGLEFINANCE to populate, then Refresh
